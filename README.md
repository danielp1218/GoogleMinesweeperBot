# GoogleMinesweeperBot
Bot to beat google minesweeper!

Currently only works on medium difficulty with 1980x1080 screen.

## How to use
1. Open google minesweeper
2. Click play
3. Run the script

## Requirements
- Python 3
- PyAutoGUI
- Pillow
- Numpy
- scikit-image
- win32gui
